### Play a fan-made remake of the original Gwent minigame against other players

**Gwent Online** is a fan-made browser port of **Gwent** from *The Witcher 3: Wild Hunt*, designed for **two-player online matches**.

## Features

- **Two-player multiplayer** – play against friends in real-time.  
- **Lobby system with join codes** – players share a unique code to connect and start a match.
- **Cards from The Witcher 3: Wild Hunt base game** - Northern Realms, Nilfgaardian Empire, Monsters and Scoia'tael decks.
- **Authentic Gwent rules** – follows the original game mechanics, including rounds and card strength scoring.  
- **Browser-based gameplay** – no installation required, play directly from your web browser.  

## How to Play

1. Host the game locally or deploy it using the provided API and WebAssembly build.  
2. Share your lobby join code with your friend.  
3. Both players connect using the code to enter the same lobby.  
4. Start the match and play using the same rules as the original Gwent.
